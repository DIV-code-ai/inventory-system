Inventory Management System

A simple web-based inventory tracker that allows users to add and manage products with their quantities. This project focuses on basic interactivity and clean UI using Bootstrap and vanilla JavaScript.

---

Features

* Add products with name and quantity
* Display products in a structured table
* Delete products from the list
* Instant updates without page reload
* Responsive layout using Bootstrap

---

Tech Stack

* HTML
* CSS (with Bootstrap)
* JavaScript (Vanilla)

---

How to Use

1. Enter a product name
2. Enter the quantity
3. Click "Add" to insert the product into the table
4. Click "Delete" to remove a product from the list
